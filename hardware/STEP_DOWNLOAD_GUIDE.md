# STEP File Download Guide — WX-Level & WX-Flow

Every component needs a real 3D STEP model to replace the placeholder boxes/cylinders
in the Fusion 360 assembly scripts. This guide lists where to download each one.

**All downloads come from exactly 4 sites** (all free accounts):
- **GrabCAD** (grabcad.com) — most mechanical parts
- **SnapEDA** (snapeda.com) — ICs and sensors
- **Ultra Librarian** (app.ultralibrarian.com) — ICs with STEP
- **TraceParts** (traceparts.com) — connectors

Every link below goes to one of these 4 sites. No other accounts needed.

---

## How to import STEP files into Fusion 360

1. Open your assembly document (the one generated by `wx_level_full_assembly.py` or `wx_flow_full_assembly.py`)
2. **Insert → Insert Mesh** or **Insert → Insert from file** → select the `.step` file
3. Position it where the placeholder body is (use Move/Copy)
4. Delete the placeholder body
5. Repeat for each component

## WORKFLOW SUMMARY

1. **Create free accounts** on:
   - GrabCAD (grabcad.com)
   - SnapEDA (snapeda.com)
   - Ultra Librarian (app.ultralibrarian.com)
   - TraceParts (traceparts.com)

2. **Download 22 STEP files** from those same 4 sites (numbered 1–22 above)

3. **Open the Fusion assembly** (run `wx_level_full_assembly.py` or `wx_flow_full_assembly.py`)

4. **For each placeholder body:**
   - Insert → Insert from file → pick the STEP
   - Move/align it to match the placeholder position
   - Delete the placeholder body
   - Apply appropriate material/appearance

5. **Model the 6 custom probe components** from the dimensions above

6. **Render** using Fusion's Render workspace → In-Canvas Render for photorealistic output
